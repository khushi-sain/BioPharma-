# 💊 Online Pharmacy Management System

An interactive and robust full-stack web application designed to digitize the pharmaceutical experience. This system bridges the gap between healthcare providers and patients by providing a seamless platform for medicine procurement, prescription management, and real-time inventory tracking.

---

## 🚀 Key Features

### 👤 User Features
*   **Smart Search:** Filter medicines by category, brand, or salt composition.
*   **Prescription Gateway:** Securely upload doctor prescriptions for pharmacist verification.
*   **Order Tracking:** Real-time status updates from "Processing" to "Delivered."
*   **Profile Management:** Maintain a history of previous medical orders and invoices.

### 🛡️ Admin & Pharmacist Features
*   **Precision Inventory Intelligence:** Automated alerts for "Low Stock" and "Near Expiry" items.
*   **Role-Based Access:** Distinct dashboards for Admin (Sales/Analytics) and Pharmacist (Order fulfillment).
*   **Automated Billing:** Instant PDF invoice generation for every transaction.
*   **Data-Driven Insights:** Visual reports of daily and monthly sales trends.

## 🛠️ Tech Stack

*   **Frontend:** HTML5, CSS3, JavaScript (Modern & Responsive UI)
*   **Backend:** PHP / Node.js (As per your implementation)
*   **Database:** MySQL (Relational Data Management)
*   **Tools:** VS Code, Git, XAMPP/WAMP

---

## 🎯 Project Objectives

1.  **Bridging the Healthcare Gap:** Providing 24/7 access to essential medications.
2.  **Zero-Error Processing:** Eliminating manual errors in billing and prescription reading.
3.  **Data Integrity:** Ensuring 100% secure storage of patient records and transaction history.
4.  **Operational Agility:** Reducing the order-to-delivery cycle time through automation.
